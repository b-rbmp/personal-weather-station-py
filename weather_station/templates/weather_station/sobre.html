Sobre

Este website implementa uma aplicação de Internet Das Coisas agregando dados de "estações metereológicas" realizadas com o uso de microcontroladores conectados a internet. Os dados colhidos são Temperatura, Luminosidade, Humidade e Pressão, através de sensores integrados aos microcontroladores

O envio dos dados é realizado remotamente através de comunicação HTTP entre os microcontroladores e o servidor, que realiza a sua devida armazenagem num banco de dados. Os microcontroladores são identificados através de uma chave que deve ser previamente registrada no servidor de forma a aceitar a submissão de dados

O servidor foi realizado com o uso da framework Django (python), enquanto que os microcontroladores podem ser de qualquer tipo, contanto que estejam conectados a internet e possam enviar requisições HTTP. Os exemplos mostrados são realizados com microcontroladores ESP8266

Mais detalhes podem ser visto no repositório GitHub: Repositório
